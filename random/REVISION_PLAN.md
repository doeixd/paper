### 4. Section 9.4 - Ethics (Lines 998-1095)
**Issue**: Risks appearing to derive ought from is despite recent additions
**Changes** (drawing from proc_v7.md):
- **Add two-part defense structure** after "Brave New World" paragraph:
  - **Constitutive Defense**: "The Persistence Condition functions as a methodological constraint, not a normative commitment. Any normative system that fails to persist becomes unavailable for analysis. We describe the filtering process that operates on persistent systems without claiming persistence is intrinsically valuable."
  - **Instrumental Defense**: "The framework offers conditional oughts: 'If a system aims to persist while enabling cooperation, then it should adopt low-brittleness principles.' For systems indifferent to persistence, our framework provides no normative force."
- **Add independence of constraints** paragraph: "The pragmatic constraints that filter normative systems—biological needs, cognitive limitations, coordination requirements—are empirically discoverable facts, not value judgments. These constraints are what filter systems, not products of the filtering process itself."
- **Strengthen floor/ceiling distinction**: Make it even more explicit that framework maps viability necessities, not flourishing sufficiency
